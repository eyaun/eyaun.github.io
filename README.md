# Personal Portfolio Website

A modern, responsive personal portfolio website designed to showcase my projects, technical skills, and professional background as a developer.

This website serves as a central hub where visitors, recruiters, and collaborators can explore my work, learn about my expertise, and easily get in touch with me.

---

## Project Overview

This portfolio website was created to demonstrate my web development skills and to present my projects in a clear and professional manner. The website highlights selected projects, technical competencies, and personal information while maintaining a clean and user-friendly design.

The design emphasises simplicity, performance, and accessibility so that visitors can quickly understand who I am and what I build.

---

## Features

- Clean and modern user interface
- Fully responsive design (mobile, tablet, desktop)
- Project showcase section
- Skills and technology stack display
- Professional contact section
- Structured navigation
- Optimised layout for readability
